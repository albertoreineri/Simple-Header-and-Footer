Simple Header and Footer Plugin let you easily insert code in the header (between <head> and </head> tags) and in the footer (before the </body> tag). Ideal for inserting Analytics scripts or other tools easily and quickly.

== Description ==

Simple Header and Footer Plugin allows you to add Google Analytics, Facebook Pixel, custom tracking code, and more with a straightforward interface. Avoid installing complex plugins with thousands of unnecessary functionalities

= Translation =

You can contribute to translate this plugin in your language on [WordPress Translate](https://translate.wordpress.org)

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
